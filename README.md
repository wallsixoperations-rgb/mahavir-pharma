# Mahavir Pharma – Order Management System

## Overview
A minimal web-based order management system built to efficiently handle and track new orders for Mahavir Pharma.

## Features
- Add new orders
- View and manage order list
- Simple and clean interface
- Fast and lightweight

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
Built for internal use to streamline daily order management and improve operational efficiency.
